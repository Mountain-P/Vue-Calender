# 行事曆(Vue)

## 技術支持
```
前端:Vue
框架:Vuetufy
後端(網頁伺服器):Express
資料庫:Sqlite3
```

### NPM安裝
```
npm install
```

### NPM編譯
```
npm run build
```

### 開始
```
npm run start
```
### 打開瀏覽器並輸入以下網址
```
localhost:8080
```
